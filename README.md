# transcoder
Make transcoder with FFmpeg

## Features
- [ ] Poll jobs from HTTP(s) or File endpoints
- [ ] Update encoding status to HTTP(s) or File endpoints
- [ ] Stop jobs when job.enable is false
- [ ] Pre-start, inited, Post-end, segmented callback
- [ ] metadata of input and output files
- [ ] preview images
- [ ] POST job in AWS SQS via AWS API Gateway
- [ ] Update job status in AWS DynamoDB via AWS API Gateway
- [ ] Browse Files stored in S3 via AWS API Gateway
- [ ] Trigger job publish to SQS and launch EC2 instance on new files created in S3
- [ ] Web-based file uploading page for S3
